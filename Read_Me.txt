The Game Lair of Shadows was created
for our CECS 343 (Introduction to Software
Engineering) class at California State University,
Long Beach (CSULB).

The team, ShadowTech, is compromised of four
members, they are:

Beau Squires
Eduardo Arevalos
Sage Hammons
Christian Ovid

The game is text based role-playing/adventure game implemented
using a Graphical User Interface (GUI). The GUI was created by 
implementing the java.swing and java.awt packages of the java libraries
